### 🧑‍💻 About Me

I'm a **Computer Science Engineering student** and aspiring full-stack developer with a deep interest in **Artificial Intelligence**, **Data Analytics**, and **IoT systems**. I love turning ideas into real, deployable products — from AI-powered web apps to hardware prototypes.

- 🔭 Currently building AI-powered applications using **React / Next.js** + **Python backends**
- 🤖 Exploring **Hugging Face models**, **Groq inference**, and **open-source AI tooling**
- 📊 Passionate about turning raw data into meaningful insights
- 🛠️ Experience with **Arduino & embedded systems** for real-world IoT solutions
- 🚀 Deploying projects at **zero cost** using Vercel, Hugging Face Spaces & Supabase
- 🌱 Always learning, always shipping
